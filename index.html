Robot Localization

Robot localization is the method a robot uses to determine where it is. This can be tricky to implement because it's possible for a sensor to return an erroneous reading. An erroneous reading means a robot thinks it's in the wrong location and this could be disastrous (imagine if a self-driving car didn't know where it was!). One method to account for this is a particle filter. This is an implementation and visualization of this method.
